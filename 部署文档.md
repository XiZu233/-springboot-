# 图书管理系统 - 零基础部署运行手册

本手册面向没有 Java / MySQL 基础的用户，介绍如何在 Windows 电脑上从零开始安装环境、部署并运行本图书管理系统。

---

## 一、分发包目录说明

将压缩包解压后会得到以下主要目录和文件：

```text
book-management/               # 解压 release/book-management.zip 后得到的文件夹
├── 安装包/                    # 运行本项目所需的环境安装程序
│   ├── OpenJDK17U-...msi     # JDK 17 安装包（运行 Java 程序必需）
│   ├── mysql-installer-...msi # MySQL 8.0 安装包（数据库必需）
│   └── 7z2408-x64.exe        # 7-Zip 解压工具（解压项目压缩包用）
│
├── book-management.sql        # 数据库初始化脚本
├── 部署文档.md                 # 本手册
├── pom.xml                    # Maven 构建配置
├── src/                       # 项目源代码
└── target/                    # 编译产物（含可运行 jar）
```

## 二、环境要求

| 软件 | 版本要求 | 用途 |
| --- | --- | --- |
| JDK | 17 及以上 | 运行 Spring Boot 程序 |
| MySQL | 5.7 或 8.0 | 存储用户和图书数据 |

## 三、安装运行环境

### 3.1 安装 JDK 17

1. 打开 `安装包/` 文件夹。
2. 双击运行 `OpenJDK17U-jdk_x64_windows_hotspot_17.0.13_11.msi`。
3. 按照向导点击 **Next** → **Next** → **Install** → **Finish**，全部使用默认选项即可。
4. 安装完成后，按 `Win + R`，输入 `cmd` 回车，打开命令提示符。
5. 输入以下命令验证安装：

   ```cmd
   java -version
   ```

   如果看到类似 `openjdk version "17.0.13"` 的信息，说明安装成功。

### 3.2 安装 MySQL 8.0

1. 打开 `安装包/` 文件夹。
2. 双击运行 `mysql-installer-community-8.0.40.0.msi`。
3. 选择安装类型：
   - 初学者建议选择 **Server only**（仅安装数据库服务器）。
4. 一路点击 **Next** / **Execute**。
5. 在 **Authentication Method** 页面，选择 **Use Strong Password Encryption**（推荐）。
6. 在 **Accounts and Roles** 页面，设置 **Root 密码为 `123456`**（本系统默认使用该密码）。
7. 在 **Windows Service** 页面，保持默认勾选 **Configure MySQL Server as a Windows Service**，服务名建议为 `MySQL80`。
8. 点击 **Execute** 完成安装。
9. 安装完成后，MySQL 会自动作为 Windows 服务启动。后续开机后会自动运行。

> 验证方式：按 `Win + R` 输入 `services.msc`，找到 `MySQL80` 服务，状态为 **正在运行** 即可。

### 3.3 安装 7-Zip

1. 打开 `安装包/` 文件夹。
2. 双击运行 `7z2408-x64.exe`。
3. 点击 **Install** 完成安装。

## 四、部署项目

### 4.1 解压项目

1. 右键点击 `book-management.zip`。
2. 选择 **7-Zip** → **提取到当前位置**（或 **Extract Here**）。
3. 解压后得到 `book-management/` 文件夹，这就是完整的项目目录。

### 4.2 初始化数据库

1. 按 `Win + R`，输入 `cmd` 回车，打开命令提示符。
2. 使用 `cd` 命令进入解压后的项目目录，例如：

   ```cmd
   cd /d E:\book-management
   ```

   > 注意：请根据实际解压路径修改。

3. 执行数据库初始化脚本：

   ```cmd
   mysql -uroot -p123456 < book-management.sql
   ```

   执行时不会显示任何提示，等待命令行重新出现即表示完成。

4. 验证数据库：

   ```cmd
   mysql -uroot -p123456 -e "USE book_management; SHOW TABLES;"
   ```

   应输出 `books` 和 `users` 两张表。

### 4.3 启动图书管理系统

在项目目录的命令行中执行：

```cmd
java -jar target/book-management-0.0.1-SNAPSHOT.jar
```

看到类似以下日志即表示启动成功：

```text
Started BookManagementApplication in x.xxx seconds
Tomcat started on port(s): 8080 (http)
```

> 启动成功后，**不要关闭** 这个黑色命令窗口，否则系统会停止运行。

## 五、使用系统

### 5.1 登录

1. 打开浏览器（Chrome / Edge / Firefox 均可）。
2. 访问地址：

   ```text
   http://localhost:8080
   ```

3. 会自动跳转到登录页。
4. 输入默认管理员账号：
   - **账号：`admin`**
   - **密码：`123456`**
5. 点击 **登录**。

### 5.2 添加图书

1. 登录后进入图书列表页。
2. 点击 **+ 添加图书**。
3. 填写图书信息：
   - 图书名称（必填）
   - 类别
   - 作者
   - 出版社
   - 版本
4. 点击 **保存**，系统会自动返回列表页并显示新添加的图书。

### 5.3 删除图书

1. 在图书列表页找到要删除的图书。
2. 点击该行右侧的 **删除** 按钮。
3. 在弹出的确认框中点击 **确定**。
4. 图书会从列表中移除。

### 5.4 注册新用户

1. 在登录页点击 **立即注册**。
2. 输入用户名、密码和确认密码。
3. 点击 **注册**。
4. 注册成功后会跳回登录页，使用新账号登录即可。

> 普通用户登录后同样可以进行图书的增删查操作。

### 5.5 退出登录

点击页面右上角的 **退出登录** 按钮即可。

## 六、验证数据库同步

添加或删除图书后，可以在 MySQL 中查看数据是否同步：

```cmd
mysql -uroot -p123456 --default-character-set=utf8mb4 -e "USE book_management; SELECT * FROM books;"
```

网页端添加的图书会立即出现在查询结果中；删除后则查询结果为空。

## 七、停止系统

关闭运行 `java -jar ...` 的黑色命令窗口，即可停止图书管理系统。

MySQL 作为 Windows 服务会保持运行，不影响其他程序。如需停止，可在 `services.msc` 中右键 `MySQL80` 选择 **停止**。

## 八、常见问题

### Q1：提示 "'java' 不是内部或外部命令"

JDK 安装后没有自动配置环境变量。请重新安装 JDK，或在安装时勾选 **Set JAVA_HOME variable** 选项。

### Q2：提示 "Can't connect to MySQL server"

MySQL 服务没有启动。按 `Win + R` 输入 `services.msc`，找到 `MySQL80` 服务并右键选择 **启动**。

### Q3：忘记 MySQL root 密码

本系统默认使用 root / 123456。如果密码不同，需要修改 `src/main/resources/application.yml` 中的 `password` 配置，然后重新打包：

```cmd
mvn clean package -DskipTests
```

### Q4：如何修改项目代码后重新打包

1. 修改代码。
2. 安装 Maven（官网下载地址：`<https://maven.apache.org/download.cgi>`）。
3. 在项目目录执行：

   ```cmd
   mvn clean package -DskipTests
   ```

4. 重新运行生成的 jar 文件。
